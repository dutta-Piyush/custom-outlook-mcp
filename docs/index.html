outlook-mcp

Blazing-fast MCP server for Microsoft Outlook via Graph API. 8 tools — search, read, send, reply — all from VS Code chat. Built in Go for instant startup.

GitHub ↗ | Graph Explorer ↗ | Go 1.23+ MCP v1.4 ~30ms cold start

Quick Start

1. Get a Microsoft Graph Token

Go to Graph Explorer, sign in with your corporate account, and copy the access token. Required scopes: Mail.ReadWrite, Mail.Send.

2. Clone & build

$ git clone https://github.deutsche-boerse.de/at384/outlook-mcp.git
$ cd outlook-mcp
$ go build -o outlook-mcp.exe .

If you have SSH configured: git clone git@github.deutsche-boerse.de:at384/outlook-mcp.git

# Single binary, zero runtime dependencies
outlook-mcp.exe — 11.4 MB — ready

3. Add to VS Code

Add to %APPDATA%\Code\User\mcp.json:

{
  "servers": {
    "outlook": {
      "command": "C:/path/to/outlook-mcp.exe",
      "env": { "OUTLOOK_TOKEN": "your-graph-token" },
      "type": "stdio"
    }
  }
}

4. Reload VS Code

Ctrl+Shift+P → Developer: Reload Window. The outlook server appears in your MCP list. Done.

Why Use This

Lightning-fast search

KQL-powered full-text search across your entire mailbox. Find any email in milliseconds using from:, subject:, date ranges, and boolean operators.

Send & reply from chat

Draft and send emails, reply to threads, CC/BCC — all without leaving VS Code. Copilot picks the right tool automatically.

Zero startup overhead

Single Go binary. ~30ms cold start vs ~2s for Python servers. HTTP/2 connection pooling to Graph API keeps subsequent calls instant.

Powerful filtering

OData $filter for structured queries (unread, importance, date ranges), KQL $search for free-text. Pagination, custom field selection, folder scoping.

Architecture

 ┌─────────────┐    stdio/JSON-RPC    ┌─────────────────┐   HTTPS + Bearer   ┌────────────────┐
 │  VS Code    │ ◄──────────────────► │  outlook-mcp    │ ◄───────────────► │  Microsoft     │
 │  Copilot    │                      │  (Go binary)    │   HTTP/2 pooled    │  Graph API     │
 └─────────────┘                      └─────────────────┘                    └────────────────┘

Transport

stdio — VS Code spawns the Go binary, communicates via JSON-RPC over stdin/stdout. Zero network hop.

Auth

Bearer token (delegated) via OUTLOOK_TOKEN env var. Get from Graph Explorer or MSAL.

Stack

Go 1.23 + github.com/modelcontextprotocol/go-sdk (MCP v1.4) + net/http with HTTP/2 connection pooling.

Troubleshooting

Server not showing?

Ctrl+Shift+P → MCP: Reset All Cached MCP Servers, then reload.

401 Unauthorized?

Ensure your token has Mail.ReadWrite and Mail.Send scopes.

$search errors?

KQL search cannot combine with $orderby. Use $filter + orderby for sorted results, or $search alone for relevance-ranked results.
